Detailed Description

Allows an Animal to move.

Attach this script to any GameObject that has an Animal script attached.
This will allow for linear movement, as well as following a SplineMovement.
It also has many helpful features, such as the MoveWithGround property, which will allow an Animal to stick to the current Collider the Animal is sitting on.
