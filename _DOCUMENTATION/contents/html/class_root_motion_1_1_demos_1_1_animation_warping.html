Detailed Description

Warping an effector from animation space to world space. The weight curve of the warp is used to add the offset from "Warp From" to "Warp To" to the effector. "Warp From" should be a Transform parented to the root of the character, hence in animation space (virtual position where a soccer player's foot hits the ball in the animation). "Warp To" should be a Transform in the world space (the actual ball).
