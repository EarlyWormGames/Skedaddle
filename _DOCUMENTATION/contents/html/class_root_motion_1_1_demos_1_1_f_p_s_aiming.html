Detailed Description

Basic full body FPS IK controller.

If aimWeight is weighed in, the character will simply use AimIK to aim his gun towards the camera forward direction. If sightWeight is weighed in, the character will also use FBBIK to pose the gun to a predefined position relative to the camera so it stays fixed in view. That position was simply defined by making a copy of the gun (gunTarget), parenting it to the camera and positioning it so that the camera would look down it's sights.
