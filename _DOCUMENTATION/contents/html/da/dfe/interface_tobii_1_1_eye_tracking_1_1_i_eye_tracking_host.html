IDataProvider<EyePositions> Tobii.EyeTracking.IEyeTrackingHost.GetEyePositionDataProvider ( )

EXPERIMENTAL Gets a provider of eye position data. See IDataProvider<T>.

Returns
The data provider.

This data stream should be considered experimental. It might be removed or changed in a future release. We have not been able to find any kind of proper or good-enough user experience based on this data but provide it here for you to experiment with.
